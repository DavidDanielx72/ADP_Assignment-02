### Wholesale Management System (Group Project)
🔗 https://github.com/DavidDanielx72/ADP_Assignment-02

This is a backend Java system developed collaboratively as part of a group project to simulate a wholesale store management system for privately owned businesses. The system was designed to handle core business operations such as customer management, product handling, and system validation.

The project followed proper object-oriented programming principles and was structured using packages to ensure maintainability and scalability.

#### My Contribution:
- Developed the Customer class and related logic
- Ensured correct package structure across the system
- Created missing test classes to support system validation
- Assisted in debugging and ensuring system stability
- Collaborated with team members to integrate different components

#### Technologies Used:
- Java
- Object-Oriented Programming (OOP)
- IntelliJ IDEA
- GitHub (Version Control & Collaboration)
